About This Token

This project is built by an independent developer and a small team committed to testing, education, and open-source collaboration in blockchain.

While the token uses the same name and symbol as Tether USD (USDT), and a modified logo for test purposes, it is not affiliated with Tether Ltd.

The goal is to learn, experiment, and share knowledge about token deployment, metadata registration, and decentralized finance tools. No funds are being solicited, and this token is not intended for trading or real financial use.
